    Welcome

    Welcome to the Learning Lab, a platform dedicated to empowering you in the seamless integration of NeuralSeek with your selected KnowledgeBase and Virtual Agent. Throughout this lab, you will learn to harness NeuralSeek's sophisticated AI capabilities, which will elevate your virtual agent to the forefront of natural language generation.

    Unlock the full potential of AI with no code, minimal effort and all the necessary enterprise functionality and security.

    Challenges

    Participants will walk through step-by-step tutorials on fine-tuning AI-generated answers using the tools and metrics on the NeuralSeek user interface, and explore the powerful mAIstro playground which is #1 in retrieval augmented generation for enterprise. By seamlessly incorporating NeuralSeek, you'll discover how to ensure that your AI responses not only uphold accuracy but also amplify scalability, all while retaining the essential element of human oversight.

    Challenge 1: Seeking Answers

    This module showcases some of NeuralSeek's pivotal features: Semantic Scoring, Virtual Agent training, and PII filters for user privacy, all with built-in tools for human oversight to maintain data integrity.

    • Level: Entry
    • Time: 20 minutes
    • No Code Required

    Challenge 2: mAIstro Essentials

    This module offers a look inside our dynamic content creation and retrieval platform, featuring advanced data quality enhancement tools and connectors to facilitate seamless interaction with Large Language Models for refined content generation without coding.

    • Level: Intermediate
    • Time: 20 minutes
    • No Code Required

    Challenge 3: Mastering mAIstro

    This module is designed to help business users confidently begin building and scaling custom AI Agents, with tools like the Auto-Builder offering a customizable starting point - all with no-code.

    • Level: Advanced
    • Time: 20 minutes
    • No Code Required

    Get Certified

    NeuralSeek Learning Lab participants can recieve an official NeuralSeek Certification upon completion.

    How To Get Certified

    1. Participants will set up a NeuralSeek instance. NeuralSeek is available through your cloud provider of choice, or through a 1-week free trial on AWS Marketplace. See the section below for steps to provision a NeuralSeek instance.
    2. Participants will walk through each challenge. These are self-paced tutorials that guide users through the NeuralSeek platform and it's many levers and capabilities.
    3. Participants will take the quiz in the Get Certified section of the Learning Labs. Users will be prompted to submit their name and email at the end of the quiz to earn their NeuralSeek Certification.
    4. Participants will recieve an email with additional steps on how post their NeuralSeek Certification to LinkedIn, and how they can get in touch with a NeuralSeek expert for an free discovery session.

    Setting up NeuralSeek

    Before participating in this Learning Lab, it is essential that users come prepared with the necessary resources to fully engage in the practical exercises. Participants can access a 1-week free trial of NeuralSeek through AWS Marketplace. Watch the video below for instructions on how to launch your free trial. Otherwise, follow the steps below to access a paid NeuralSeek instance on your cloud provider of choice.

    Warning

    Please be aware that any purchases or subscriptions related to the aforementioned tools must be made independently by the users. It is the responsibility of each participant to understand and manage the associated costs related to the use of NeuralSeek and any other integrated services within their IBM Cloud environment. Kindly ensure that all necessary resources are set up and accessible prior to beginning the Learning Lab for a seamless and enriching experience.

    NeuralSeek Free Trial Tutorial

    IBM Cloud

    • Active NeuralSeek subscription. We recommend the Pay-per-Answer plan.
      • If BYOLLM plan is selected, we recommend the watsonx platform Large Language Model (LLM) "Llama-2-chat 70B".
    • KnowledgeBase, we recommend the IBM KnowledgeBase "Watson Discovery".
    • Virtual Agent, we recommend the IBM Virtual Agent "watsonx Assistant".
    Steps to Set Up NeuralSeek

    Log in to IBM Cloud.

    Search and select NeuralSeek from Catalog.

    • (a) Select the Pay-per-answer plan type.
    • (b) To agree to the third-party terms, in the bottom right, open and read the terms then check, “I have read and agree to the following third-party terms”.
    • (c) To provision, click “Create”.
    • (d) Click “Launch NeuralSeek”.

    Warning

    Please note that users are responsible for all incurred costs.

    AWS Marketplace

    • Active NeuralSeek subscription: we recommend the Pay-per-Answer plan.
      • If BYOLLM plan is selected, we recommend the Amazon Bedrock Large Language Model (LLM)"Claude 3 Haiku."
    • KnowledgeBase: we recommend the AWS KnowledgeBase "Kendra Index."
    • Virtual Agent: we recommend the AWS Virtual Agent "LexV2 Lambda."
    Steps to Set Up NeuralSeek

    Log in to the AWS Console Marketplace page.

    • (a) Click the yellow Manage subscriptions button.
    • (b) On the left side menu, click Discover products.
    • (c) In the Search AWS Marketplace products enter NeuralSeek and press enter. You will see two products in the search result.
      • If you do not have access to your own large language model (LLM), you may choose the first one which is NeuralSeek, which does provide our curated LLM included in it.
      • If you do have your own LLM, and thus have the necessary keys and region, (role access key, secret access key, and region) you can choose NeuralSeek - Bring Your Own Large Language Model.
    • (d) After you select a product from the list, click the View purchase options button at the top right.
    • (e) Review the offer, and pricing information. When ready click the Subscribe button under Purchase.

    Warning

    Please note that these NeuralSeek subscriptions, when created, will incur billing and cost as they are used.

    Go back to your Manage subscriptions page.

    You may either have NeuralSeek or NeuralSeek - Bring Your Own Large Language Model depending on what you chose to subscribe for the lab.

    • (a) Click Set up Product in the bottom right corner to enter the product setup page.
    • (b) Click Set up your account in the top right corner to enter the NeuralSeek instance control page.

    NeuralSeek's instance control page should be initially empty. Here, we need to:

    1. Create a user for accessing NeuralSeek instance
      • (a) Enter email address into Add User (Email) box.
      • (b) Click on the plus icon to add a user.
      • (c) Newly created user’s email will populate below.
      • (d) A new password will be generated. Passwords can be changed later using the password reset functionality.
    2. Create an instance of NeuralSeek
      • Generating an instance at this stage will not initiate any server activity. Creating an instance merely establishes a unique identifier within the database, enabling authentication and access to the system.
      • (a) Select desired location in the Create Instance box. For this lab, we recommend US-West-2.
      • (b) Click the plus icon to create an instance.
      • (c) Click the writing tool icon.
      • (d) Enter desired instance name into text box.
      • (e) Click the writing tool icon to save changes.
      • (f) Updated instance name will populate.
    3. Add created user to the created NeuralSeek instance
      • (a) Click Configure User Access to Instances plus icon.
      • (b) Click the plus icon.
      • (c) User is now able to connect to the instance.
      • (d) Click the instance link learning-lab-test in the list of instances.

    Fill out the necessary information below to log in to NeuralSeek.

    • (a) Enter user email address.
    • (b) Enter user password generated from the Create a User section. If desired, you are able to reset the password using the Forgot password? link.
    • (c) Click Continue to be directed to the NeuralSeek user interface page.

    Azure Marketplace

    • Active NeuralSeek subscription: we recommend the Pay-per-Answer plan.
      • If BYOLLM plan is selected, we recommend the OpenAI Large Language Model (LLM)"GPT-4o"
    • KnowledgeBase: we recommend the Azure supported KnowledgeBase "ElasticSearch"
    Steps to Set Up NeuralSeek

    Log in to the Azure Marketplace and sign in.

    • (a) Search for the NeuralSeek product page.
    • (b) Click Get It Now.
    • (c) If prompted, fill out the additional information, such as name, email, and country/region.

    You will be redirected to your Microsoft Azure Portal account. Be sure to sign in.

    • (a) Click the Subscribe button.
    • (b) Turn off recurring billing.
    • (c) Review and click subscribe.
    • (d) Configure your account.
    • (e) Activate your account.

    You will be directed to the NeuralSeek Instance Control page.

    • (a) Create an instance.

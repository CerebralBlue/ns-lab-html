    1.2 - Configurar NeuralSeek

    Básicos - Primeros Pasos

    En primer lugar, necesitamos una breve descripción sobre su empresa u organización, centrándonos en el caso de uso para esta instancia de NeuralSeek.

    • (a) Agregue una breve descripción de la empresa u organización.
    • (b) Seleccione el idioma de salida en el menú desplegable. En este ejemplo, seleccione Inglés. Lo mejor es que coincida con el idioma de su base de conocimientos y preguntas, pero NeuralSeek puede traducir entre muchos idiomas.
    • (c) Seleccione el uso preferido de la instancia de NeuralSeek. ¿Se usará esta instancia de NeuralSeek de forma interna dentro de su organización o es externa y dirigida al cliente? Utilizaremos esta información para sintonizar algunos parámetros de seguridad para usted. En este ejemplo, seleccione Interno.

    ¡Haga clic en Siguiente para guardar una descripción única, un idioma y una opción de caso de uso!

    Datos - Conéctese a su Base de Conocimiento

    • (a) Seleccione la base de conocimientos deseada en el menú desplegable. Para el propósito de este laboratorio, recomendamos la base de conocimientos de AWS Kendra.
    • (b) Seleccione el idioma deseado. Para el propósito de este laboratorio, haga clic en Inglés.

    Agregar Información de la Base de Conocimientos

    El usuario es responsable de proporcionar la información detallada de la base de conocimientos seleccionada. Dado que se tarda un tiempo en configurar estos datos para usted mismo, vamos a usar uno que ya está preparado para el laboratorio de aprendizaje, con acceso de solo lectura. Los valores reales a continuación serán proporcionados durante el curso de este laboratorio por el instructor del laboratorio.

    ⚠️ Para entender cómo crear y localizar las claves de acceso de AWS, consulte la sección Información adicional - Claves de acceso en la guía del laboratorio.

    Por favor, ingrese lo siguiente:

    • (a) Agregar ID de índice Kendra.
    • (b) Agregar Clave de acceso de rol de AWS.
    • (c) Agregar Región de AWS.
    • (d) Agregar Clave de acceso secreta de rol de AWS.

    Haga clic en el botón Probar para probar la conexión. Luego, haga clic en Siguiente para guardar la configuración de la base de conocimientos.

    ¡NeuralSeek ahora está configurado con una base de conocimientos, en este caso: índice Kendra, y listo para buscar!

    ⚠️ Para crear su propio índice de Kendra para hacer el laboratorio

    Con el interés del tiempo, el laboratorio usará un índice Kendra ya incorporado que está prellenado con los datos. Eso se debe al hecho de que crear el índice Kendra y poblar los datos tardaría de 40 minutos a más de una hora en completarse. Sin embargo, en caso de que esté interesado en realizar más tarde el laboratorio usando su propio índice Kendra y claves, puede seguir las siguientes instrucciones.

    Organiza - Resultados y Categorías

    Categorización

    Categoriza tus intenciones para facilitar el informe y la gestión.

    Si está siguiendo el laboratorio en vivo, ¡saltee este paso! Volveremos a este tema en el Módulo 2.

    • (a) Haga clic en el ícono de la bombilla para agregar una nueva fila.
    • (b) Ingrese un nombre de categoría único, una URL correspondiente y una descripción detallada de la categoría con intenciones que no coincidan con ninguna otra categoría.

    Marco de Agente Virtual

    • (a) Seleccione el tipo de agente virtual en el menú desplegable. En este ejemplo, seleccione AWS Lex V2.
    • (b) Seleccione la opción preferida para habilitar o deshabilitar la inclusión de enlaces incrustados en las respuestas devueltas. En este ejemplo, seleccione Deshabilitar.

    Haga clic en Siguiente para guardar las categorías informadas y el marco de agente virtual seleccionado.

    Afinar - Afinar para sus datos

    Afinemos NeuralSeek para sus datos.

    • (a) Seleccione la opción preferida para cuál de estos se parece más a sus datos. Para este ejemplo, seleccione Las frases importantes están rodeadas por muchas líneas de texto que las explican.
      • La opción Las frases importantes están rodeadas por muchas líneas de texto que las explican, describe datos con explicaciones más elaboradas con pasajes de texto más extensos para una mejor comprensión.
      • La opción Las frases importantes están rodeadas por solo una o dos líneas de información relevante, describe datos con una cobertura concisa de frases clave con solo una o dos líneas de contexto necesario.
    • (b) Seleccione la opción preferida que mejor describa sus requisitos de datos. Para este ejemplo, seleccione Quiero dar mucha información al LLM y darle margen de maniobra para decidir la respuesta.
    • (c) Seleccione la opción preferida que mejor describa su moneda de datos. Para este ejemplo, seleccione Todos mis documentos son actuales y relevantes.
      • La opción Todos mis documentos son actuales y relevantes describe datos que están actualizados.
      • La opción Tengo muchos documentos antiguos o publicaciones de blog, algunos con información contradictoria. Necesito que NeuralSeek me ayude con la priorización de fechas, describe datos que pueden tener detalles contradictorios o irrelevantes en diferentes fuentes o documentos.

    Haga clic en Siguiente para guardar las configuraciones de ajuste informadas.

    Pregunta y Respuesta

    A medida que NeuralSeek responde las preguntas, podemos construir y mantener automáticamente partes de su Agente Virtual.

    Utilice los botones a continuación para generar rápidamente contenido de preguntas y respuestas para arrancar y probar su Agente Virtual.

    • Generar Preguntas Automáticamente consultará la base de conocimientos, pedirá a NeuralSeek que genere información y producirá una variedad de preguntas de prueba basadas en el contenido corporativo. Haga clic en Enviar para que NeuralSeek genere respuestas.

    • Ingresar Preguntas Manualmente proporcionará un cuadro de prueba en blanco para que ingrese preguntas, una por línea, basadas en su contenido corporativo. Haga clic en Enviar para que NeuralSeek genere respuestas.

    • Cargar Preguntas de Prueba proporcionará un enlace a su explorador de archivos local para cargar un archivo CSV de preguntas de prueba. Las preguntas se ejecutarán a través del punto final Seek en paralelo y se calificarán. Utilice el archivo descargable .csv proporcionado como plantilla para sus preguntas. Los archivos de entrada deben conservar estos títulos de columna como mínimo, pero puede agregar columnas de carga útil adicionales. Haga clic en Enviar para que NeuralSeek genere respuestas.

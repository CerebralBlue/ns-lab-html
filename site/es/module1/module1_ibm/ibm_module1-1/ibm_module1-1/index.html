    1.1 - Configurar NeuralSeek

    Agregar NeuralSeek

    Inicia sesión en IBM Cloud. Busca y selecciona NeuralSeek del catálogo.

    • (a) Selecciona el tipo de plan Pago por respuesta.
    • (b) Para aceptar los términos de terceros, en la esquina inferior derecha, abre y lee los términos y luego marca "He leído y acepto los siguientes términos de terceros".
    • (c) Para aprovisionar, haz clic en "Crear".

    Tenga en cuenta que los usuarios son responsables de todos los costos que se incurran.

    Lanzar NeuralSeek

    Haz clic en "Lanzar NeuralSeek".

    Básicos - Primeros pasos

    Primero, necesitamos un breve resumen sobre su empresa u organización, centrándose en el caso de uso para esta instancia de NeuralSeek.

    • (a) Agrega una breve descripción de la empresa u organización.
    • (b) Selecciona el idioma de salida del menú desplegable. En este ejemplo, selecciona Inglés. Es mejor que coincida con el idioma de tu base de conocimientos y preguntas, pero NeuralSeek puede traducir entre muchos idiomas.
    • (c) Selecciona el uso preferido de la instancia de NeuralSeek. ¿Se utilizará esta instancia de NeuralSeek internamente dentro de su organización o es externa, orientada al cliente? Utilizaremos esta información para preconfigurar algunos parámetros de seguridad para ti. En este ejemplo, selecciona Interno.

    Datos - Conectar a tu base de conocimientos

    • (a) Selecciona la base de conocimientos. En este ejemplo, "Watson Discovery".
    • (b) Selecciona el idioma. En este ejemplo, Inglés.

    Agregar información de la base de conocimientos

    El usuario es responsable de proporcionar la información detallada de la base de conocimientos seleccionada. Como la configuración puede ser un proceso largo, utilizaremos la base de conocimientos ya preparada para este laboratorio de aprendizaje, con acceso de solo lectura. Los valores reales serán proporcionados por el instructor durante el transcurso del laboratorio.

    Por favor, ingresa lo siguiente:

    • (a) Agrega la "URL del servicio Discovery".
    • (b) Agrega la "clave API de Discovery".
    • (c) Agrega el "ID del proyecto de Discovery".

    Haz clic en el botón "Probar" para probar la conexión. Luego, haz clic en "Siguiente" para guardar la configuración de la base de conocimientos.

    Organizar - Salidas y categorías

    Categorización

    Categoriza tus intenciones para facilitar la gestión y la generación de informes.

    Si sigues el laboratorio en vivo, ¡salta este paso! Revisaremos esto en el Módulo 2.

    • (a) Haz clic en el icono de bombilla para agregar una nueva fila.
    • (b) Ingresa un nombre de categoría único, una URL correspondiente y una descripción detallada de la categoría con intenciones que no coincidan con ninguna otra categoría.

    Marco de agente virtual

    • (a) Selecciona el "tipo de agente virtual" del menú desplegable. En este ejemplo, selecciona Acciones del asistente de watsonx.
    • (b) Selecciona la opción preferida de habilitar o deshabilitar la incrustación de enlaces en las respuestas devueltas. En este ejemplo, selecciona Deshabilitar.

    Haz clic en "Siguiente" para guardar las categorías reportadas y el agente virtual seleccionado.

    Sintonizar - Ajustar para tus datos

    Ajustemos NeuralSeek para tus datos.

    • (a) Selecciona la opción preferida de ¿Cuál de estas se parece más a tus datos?. Para este ejemplo, selecciona Las frases importantes están rodeadas de muchas líneas de texto que las explican.
      • La opción Las frases importantes están rodeadas de muchas líneas de texto que las explican, describe datos con explicaciones más elaboradas con pasajes de texto más extensos para una mejor comprensión.
      • La opción Las frases importantes están rodeadas solo por una o dos líneas de información relevante, describe datos con una cobertura concisa de frases clave con solo una o dos líneas de contexto necesario.
    • (b) Selecciona la opción preferida que mejor describa tus requisitos de datos. Para este ejemplo, selecciona Quiero dar mucha información al LLM y darle margen para que decida la respuesta.
    • (c) Selecciona la opción preferida que mejor describa la actualidad de tus datos. Para este ejemplo, selecciona Todos mis documentos están actualizados y son relevantes.
      • La opción Todos mis documentos están actualizados y son relevantes, describe datos que están actualizados.
      • La opción Tengo muchos documentos antiguos o publicaciones de blogs, algunos con información contradictoria. Necesito que NeuralSeek me ayude con la priorización de fechas, describe datos que pueden tener detalles contradictorios o no relevantes entre diferentes fuentes o documentos.

    Haz clic en "Siguiente" para guardar las configuraciones de ajuste reportadas.

    Preguntas y respuestas

    A medida que se responden las preguntas por NeuralSeek, podemos construir y mantener automáticamente partes de tu agente virtual.

    Usa los botones a continuación para generar rápidamente contenido de preguntas y respuestas para agilizar el proceso y probar tu agente virtual.

    • Generar preguntas automáticamente consultará la base de conocimientos, pedirá a NeuralSeek que genere ideas y produzca una variedad de preguntas de prueba basadas en tu contenido corporativo. Haz clic en Enviar para que NeuralSeek genere respuestas.

    • Ingresar manualmente preguntas proporcionará un cuadro de texto en blanco para que ingreses preguntas, una por línea, basadas en tu contenido corporativo. Haz clic en Enviar para que NeuralSeek genere respuestas.

    • Cargar preguntas de prueba proporcionará un enlace a tu explorador de archivos local para cargar un archivo CSV de preguntas de prueba. Las preguntas se ejecutarán a través del extremo Seek en paralelo y se calificarán. Utiliza el archivo CSV descargable proporcionado como plantilla para tus preguntas. Los archivos de entrada deben conservar estos títulos de columna como mínimo, aunque puedes agregar columnas de carga útil adicionales. Haz clic en Enviar para que NeuralSeek genere respuestas.

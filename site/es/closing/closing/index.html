    After the lab is finished, it is important to make sure that the subscriptions and instances would be deleted so that any unexpected usage cost does not incur. Failing to do so may result in unnecessary billing to occur, so it is highly recommended that you would clean up the created resources after you are finished.

    Removing AWS Components

    In order to remove the subscribed NeuralSeek, please follow the below steps.

    1. Visit the AWS Marketplace subscriptions page.

    1. Click the "Manage" button under the subscription.
    2. In the Agreement section, click Actions and select Cancel subscription
    3. Read and enter confirm to the input field, and click Yes, cancel subscription.

    1. Check that the NeuralSeek subscription is no longer visible in the manage subscriptions page.
    2. Visit the Amazon Lex console.
    3. Select the bot "TestBot".
    4. Click "Action", and select "Delete". Confirm successful deletion.
    5. Visit the Functions page.
    6. Select the function "learning-lab".
    7. Click "Action", and select "Delete". Confirm successful deletion.

    Removing IBM Components

    In order to remove the NeuralSeek instance, please follow the below steps:

    1. Visit the IBM cloud resources page.
    2. Search and locate the NeuralSeek instance that you have created during the lab. Click the instance name.
    3. Select Actions and select Delete service
    4. Click OK
    5. Check that the NeuralSeek instance is no longer visible in the resources page.
    6. Search and locate the watsonx Assistant instance that you have created during the lab. Click the instance name.
    7. Select Actions and select Delete service
    8. Click OK
    9. Check that the watsonx Assistant instance is no longer visible in the resources page.

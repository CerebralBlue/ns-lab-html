    3.2 - Modelos de Exemplo

    NeuralSeek's mAIstro oferece vários modelos de exemplo adaptáveis com uma variedade de capacidades mais adequadas ao seu caso de uso.

    Carregar Modelo

    A tela introdutória do recurso mAIstro do NeuralSeek mostrará as opções para Modelos de Exemplo e Modelos de Usuário.

    • Clique em Modelos de Exemplo, em seguida, escolha a opção Escrever um Memorando.

    Se você navegou para fora da tela introdutória, clique no botão Carregar na parte inferior da tela para acessar as mesmas opções.

    Nós Preenchidos

    Dentro do Editor Visual, o fluxo de exemplo será preenchido e mostrará três nós. Vamos explorá-los mais a fundo.

    Nó de Texto

    O primeiro nó do fluxo modelo é um nó de Texto.

    • Clique na caixa do nó para mostrar o painel de propriedades. Dentro dele, há um prompt pré-escrito em linguagem natural que diz Write a memo to staff discussing the following capabilites:.

    Nó de Busca de KB

    O segundo nó do fluxo modelo é um nó de Busca de KB.

    • Clique na caixa do nó para mostrar o painel de propriedades.
    • Usando palavras-chave, execute uma consulta diretamente da base de conhecimento (KnowledgeBase) conectada. Neste exemplo, vamos substituir cognos analytics por filtering.

    Nó Enviar para LLM

    O terceiro nó do fluxo modelo é um nó de Enviar para LLM.

    Avaliar o Fluxo mAIstro

    Vamos executar o nosso fluxo!

    • Clique no botão azul Avaliar na barra inferior.
    • Observe que o conteúdo gerado aparece no painel inferior, incluindo um cabeçalho, linha de assunto, recursos-chave em forma de tópicos e alguns parágrafos de texto.

    Configurar o Nó LLM

    Clique no nó Enviar para LLM para mostrar o painel de propriedades.

    • Usando linguagem natural, opcionalmente, adicione um prompt adicional ao Modelo de Linguagem Grande (LLM). Por exemplo, adicione Limit to one paragraph.
    • Clique no botão azul Avaliar para gerar uma nova saída.

    Observe que essa saída é significativamente mais curta e formatada como um único parágrafo.

    Depurar o Fluxo

    Clique no ícone de inseto no canto superior direito para ver o caminho completo da avaliação, além de todas as variáveis de contexto.

    • Clique em cada etapa para expandir a avaliação completa.
    • Etapa #1: text
      • O prompt em linguagem natural é mostrado.
    • Etapa #2: kb
      • O conteúdo gerado mostra as informações resultantes retiradas de nossa documentação de origem com base na consulta solicitada.
    • Etapa #3: LLM
      • Observe que o conteúdo gerado aqui é o mesmo que o conteúdo resultante ao clicar no botão Avaliar. Isso ocorre porque esta é a etapa final do fluxo.
      • O conteúdo gerado mostra um memorando de um parágrafo para nossa equipe com base no conteúdo gerado na Etapa #2, assim como o prompt indica.

    Baixar Conteúdos Gerados

    Opcionalmente, você também pode baixá-los nos formatos MS Word ou CSV.

    • Clique na opção Microsoft Word ao lado de Formato de Saída para baixar o conteúdo recém-gerado como um documento MS Word.

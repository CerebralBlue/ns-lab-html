    Módulo 1: Configuração e Integração

    O objetivo do Módulo 1: Configuração e Integração é autenticar e conectar o NeuralSeek com uma Base de Conhecimento e um Agente Virtual.

    • Nível: Iniciante
    • Tempo: 20 minutos
    • Nenhum código necessário

     

    Configurando com a IBM Cloud

    Recomendamos conectar o NeuralSeek com o IBM Watson Discovery e o IBM watsonx Assistant.

    Clique aqui para começar: Módulo 1 - IBM.

     

    Configurando com o AWS Marketplace

    Recomendamos conectar o NeuralSeek com o AWS Kendra e o AWS Lex.

    Clique aqui para começar: Módulo 1 - AWS.

    1.1 - Iniciar o NeuralSeek

    Iniciar sessão

    Faça o login na página do AWS Console Marketplace page.

    Clique no botão amarelo Gerenciar assinaturas.

    Descubra produtos

    ⚠️ Se você já se inscreveu no NeuralSeek, não precisa seguir essas etapas. Por favor, vá para a seção Gerenciar assinaturas e continue.

    No menu do lado esquerdo, clique em Descobrir produtos.

    Na caixa de busca "Pesquisar produtos do AWS Marketplace", digite NeuralSeek e pressione enter. Você verá dois produtos no resultado da pesquisa:

    Selecionando o produto

    Se você não tiver acesso ao seu próprio modelo de linguagem grande (LLM), você pode escolher o primeiro que é NeuralSeek, que inclui nosso LLM curado nele.

    Se você tiver o seu próprio LLM e, portanto, tiver as chaves e a região necessárias (chave de acesso da função, chave de acesso secreta e região), você pode escolher NeuralSeek - Bring Your Own Large Language Model.

    ⚠️ Por favor, note que essas assinaturas do NeuralSeek, quando criadas, incorrerão em faturamento e custos conforme forem usadas.

    Compra

    Depois de selecionar um produto da lista, clique no botão Ver opções de compra no canto superior direito.

    Revise a oferta e as informações de preços. Quando estiver pronto, clique no botão Assinar sob Compra.

    Gerenciar assinaturas

    Volte para a sua página de Gerenciar assinaturas.

    Você pode ter NeuralSeek ou NeuralSeek - Bring Your Own Large Language Model dependendo do que você escolheu para se inscrever no laboratório.

    Configurar produto

    Clique em Configurar produto no canto inferior direito para entrar na página de configuração do produto.

    Configurar conta

    Clique em Configurar sua conta no canto superior direito para entrar na página de controle da instância NeuralSeek.

    Interface do usuário

    A página de controle da instância do NeuralSeek deve estar inicialmente vazia. Aqui, precisamos:

    1. Criar um usuário para acessar a instância NeuralSeek
    2. Criar uma instância do NeuralSeek
    3. Adicionar o usuário criado à instância do NeuralSeek criada

    Criar um usuário

    Você precisará ter uma conta de usuário para poder se conectar ao NeuralSeek.

    • (a) Insira o endereço de e-mail na caixa Adicionar usuário (e-mail).
    • (b) Clique no ícone mais para adicionar um usuário.

    Gerar senha

    • (a) O e-mail do usuário recém-criado será exibido abaixo.
    • (b) Será gerada uma nova senha. As senhas podem ser alteradas posteriormente usando a funcionalidade de redefinição de senha.

    Criar uma instância

    Gerar uma instância neste estágio não irá iniciar nenhuma atividade de servidor. Criar uma instância apenas estabelece um identificador exclusivo no banco de dados, permitindo autenticação e acesso ao sistema.

    • (a) Selecione o local desejado na caixa Criar instância. Para este laboratório, recomendamos US-West-2.
    • (b) Clique no ícone mais para criar uma instância.

    Atualizar nome da instância

    A instância recém-criada será exibida abaixo, e será gerado um ID de instância.

    • (a) Clique no ícone ferramenta de escrita.
    • (b) Insira o nome da instância desejado na caixa de texto. Para o propósito deste laboratório, recomendamos learning-lab-test.
    • (c) Clique no ícone ferramenta de escrita para salvar as alterações.
    • (d) O nome da instância atualizado será exibido.

    Conectar

    Para permitir que o usuário se conecte à instância:

    • (a) Clique no ícone de Configure User Access to Instances mais.
    • (b) Clique no ícone mais.
    • (c) O usuário agora pode se conectar à instância.

    Link da instância

    Clique no link da instância learning-lab-test na lista de instâncias.

    Fazer login

    Preencha as informações necessárias abaixo para fazer login no NeuralSeek.

    • (a) Insira o endereço de e-mail do usuário.
    • (b) Insira a senha do usuário gerada na seção Criar um usuário.

    Se desejar, você pode redefinir a senha usando o link Esqueci a senha.

    Continuar

    Clique em Continuar para ser direcionado para a página de interface do usuário do NeuralSeek.

    Um usuário ativo e uma instância do NeuralSeek agora estão configurados no console da AWS e prontos para serem conectados a uma Base de Conhecimento.

    ⚠️ Para participantes do laboratório 'NeuralSeek - Traga seu próprio Modelo de Linguagem Grande':

    Se você selecionou NeuralSeek - Bring Your Own Large Language Model e deseja usar seu próprio Modelo de Linguagem Grande para o laboratório, consulte a seção Informações adicionais - LLM do guia do laboratório para saber como configurar seu LLM no NeuralSeek.
